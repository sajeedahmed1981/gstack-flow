# Bug tracker (repo-local)

Lightweight bug log. Lives in the repo, git-versioned, no external service needed.
Mark fixed bugs done — don't delete (history is useful).

## Format
```
### BUG-NNN · <short title>   [open | in-progress | fixed | wontfix]
- **Severity:** blocker | high | medium | low
- **Found:** YYYY-MM-DD
- **Where:** file:line or URL/route
- **Symptom:** what the user sees
- **Root cause:** (fill in after investigation)
- **Fix:** commit/PR ref
```
